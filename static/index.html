Provably Fair

Verify Every Outcome

Every drill uses a commit-reveal randomness scheme. On-chain, four entropy sources are combined and hashed with SHA-256 to produce a roll between 0 and 9,999. Neither the server nor the player can manipulate the result. The API fallback uses the same logic with a pre-committed server seed hash that's revealed after each play.

Displacement

The Cascade Effect

When a position holder is defeated, they don't vanish — they drop one rank with 50% of their original stake. This can create a cascade: winning Rank 3 pushes the holder to Rank 4, potentially displacing whoever was there. The exception is Rank 10 — lose there and you're out of the pipeline entirely.

50% Stake after displacement
Rank 10 No safety net
Challenge Stake → Vault (locked)
Win Payout → 1.8x to Challenger
Loss — Defender → 10% to Defender
Loss — Remainder → 90% stays in Vault
Displacement → 50% to displaced holder
The vault retains 90% of lost stakes, creating deflationary pressure on circulating KOL.
Token Economics

Where the KOL Flows

Every token staked enters the vault. Wins pay out 1.8x from the vault. Losses split between the defender (10%) and vault retention (90%). Displaced holders receive 50% of their original stake. The vault acts as a deflationary sink — the more people play, the more KOL the vault absorbs.
