Overview

KOL is a Solana-based staking game built around a 10-position ranked pipeline. Players stake KOL tokens to "drill" into oil line positions, competing for control of the pipeline. The game uses provably-fair randomness, on-chain settlement, and a defender-rewards system to create a persistent competitive economy.

The Pipeline

The pipeline consists of 10 ranked oil line positions, each named after a historic Middle Eastern oil landmark. Higher ranks require larger stakes but yield richer rewards.

Step 1: Connect a Solana wallet (Phantom or Solflare). A player account is auto-created on first interaction.

Step 2: Select any of the 10 oil line positions. Each shows the current holder, stake, minimum requirement, and defense count.

Step 3: Enter a stake ≥ the position's minimum. Staking more increases your win chance.

Step 4: Submit the drill. Outcome is resolved via provably-fair randomness. A 30-second cooldown applies between drills.

Provably Fair System

Every drill uses a commit-reveal randomness scheme. Neither the server nor the player can manipulate the outcome.

On-Chain RNG (Primary)

SlotHashes
32 bytes — Solana sysvar
Client Seed
16 bytes — Player random
Challenger Key
32 bytes — Wallet pubkey
Timestamp
8 bytes — Unix time
SHA-256
Roll = first 2 bytes % 10,000 → (0 – 9,999)

Off-Chain RNG (API Fallback)

Server Seed
32 random bytes
Hash committed before play
Client Seed
16 random bytes
Player-generated
SHA-256(server:client)
Roll = first 8 hex chars → integer % 10,000

Verification Steps

1. Confirm SHA-256(server_seed) == server_seed_hash (proves seed wasn't changed)
2. Compute SHA-256(server_seed + ":" + client_seed)
3. Take first 8 hex characters, convert to integer, mod 10,000
4. Compare to the reported roll
5. Check roll against win threshold

Defender Economics

Holding an oil line generates passive income. Every failed drill against your position pays you 10% of the challenger's stake.

Position Displacement

When a position holder is defeated, they are displaced downward:

Rank 1-9: Defender drops to (rank + 1) with 50% of their original stake Rank 10: Defender is removed from the pipeline entirely

Background Indexer

A background process polls Solana every 15 seconds, reading all program accounts and syncing on-chain state (positions, player stats) into the local database. This keeps the API in sync with chain reality.
